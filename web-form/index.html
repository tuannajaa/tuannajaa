1 March 2020

ระบบสร้างเอกสารสำหรับการสอบสวนรับรองเรื่องต่างๆ วัตถุประสงค์เพื่อเพิ่มความถูกต้องรวดเร็วในการพิมพ์เอกสาร ลดความผิดพลาด โดยการใช้เครื่องอ่านบัตรประชาชนดึงข้อมูลและสามารถบริการผู้มาติดต่อราชการได้อย่างสะดวก รวดเร็วและมีประสิทธิภาพ

กฎหมาย ระเบียบ ที่เกี่ยวข้อง

1.การออกหนังสือรับรองการใช้อำนาจปกครองบุตรและหนังสือยินยอมให้บุตรเดินทางไปต่างประเทศ มท 0309.3/ว 20778 ลว 27 ส.ค. 2564
2.พระราชบัญญัติลักษณะปกครองท้องที่ พ.ศ. 2457
3.แนวทางการสอบสวนรับรอง
4.ระหัสสำนักทะเบียน

Download Program เพื่อดึงข้อมูลจากบัตรประชาชน

1.ติดตั้ง Chrome Extension ชื่อ Cors Unblock และเปิดใช้งาน
2.ติดตั้ง Smartcard Readder และเปิดใช้งาน
3.พิมพ์ chrome://flags/#allow-insecure-localhost ใน Browser Chrome แล้ว Enabled

4.ทดสอบ Localhost
