ระบบออกเอกสาร สอบสวนรับรอง ฝ่ายปกครอง สำนักงานเขตทุ่งครุ

1 March 2020

ระบบพิมพ์เอกสารการสอบสวนรับรองเรื่องต่างๆ วัตถุประสงค์เพื่อเพิ่มความรวดเร็ว ถูกต้อง ในการพิมพ์เอกสาร ลดความผิดพลาดในการพิมพ์ของเจ้าหน้าที่โดยการใช้เครื่องอ่านบัตรประชาชนดึงข้อมูล รวมถึงสามารถบริการประชาชนผู้มาติดต่อราชการได้อย่างรวดเร็วและมีประสิทธิภาพ

กฎหมาย ระเบียบ ที่เกี่ยวข้อง

1.การออกหนังสือรับรองการใช้อำนาจปกครองบุตรและหนังสือยินยอมให้บุตรเดินทางไปต่างประเทศ มท 0309.3/ว 20778 ลว 27 ส.ค. 2564
2.พระราชบัญญัติลักษณะปกครองท้องที่ พ.ศ. 2457
3.แนวทางการสอบสวนรับรอง
4.ระหัสสำนักทะเบียน

Download Program

1.ติดตั้ง Chrome Extension ชื่อ Cors Unblock และเปิดใช้งาน
2.ติดตั้ง Smartcard Readder และเปิดใช้งาน
3.เปิดใช้งาน Enabled chrome://flags/#allow-insecure-localhost
4.ทดสอบ Localhost
