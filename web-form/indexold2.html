รับรองสถานที่ตาย

7.1ออกหนังสือรับรองการตาย
7.2ฟอร์มบันทึกสอบสวนปากคำการเสียชีวิต ป.ค.14

หนังสือยินยอมให้บุตรเดินทางไปต่างประเทศ

1.1 บุตรไปต่างประเทศ บุตรคนเดียว
1.2 บุตรไปต่างประเทศ บุตร 2 คน
1.3 พ่อทำคนเดียว บุตรคนเดียว
1.4 แม่ทำคนเดียว บุตรคนเดียว

หนังสือรับรองสถานภาพการสมรส (รับรองโสด)

3.1 รับรองโสด (ไม่ได้จดสมรส)
3.2 รับรองโสด (จดสมรสและหย่าแล้ว)
3.3 รับรองโสด มอบอำนาจ (ไม่ได้จดสมรส)
3.4 รับรองโสด มอบอำนาจ (จดสมรสและหย่าแล้ว)

หนังสือรับรองมารดาใช้อำนาจปกครองบุตร

4.1 มารดาอุปการะบุตรเพียงผู้เดียว (ไม่ได้สมรส)
4.2 มารดาอุปการะบุตรเพียงผู้เดียว (สมรสและหย่าแล้ว)
4.3 มารดาอุปการะบุตรเพียงผู้เดียว + ยินยอมบุตรไป ตปท.

Download Program Ver.1

1.Download JCardReader

2.เปิดใช้งานโปรแกรม JHOS เพื่อดึงข้อมูลจากบัตรฯ และ ลง Program Java (หากยังไม่มี)

3.เปิด Chrome และ Enabled chrome://flags/#allow-insecure-localhost

Download Program Ver.2

1.ติดตั้ง Chrome Extension ชื่อ Cors Unblock แล้วเปิดใช้งาน

2.ติดตั้ง SmartCardReader Download แล้วเปิดใช้งานก่อนเสียบบัตร

3.Check Localhost

4.เปิด Chrome และ Enabled chrome://flags/#allow-insecure-localhost
